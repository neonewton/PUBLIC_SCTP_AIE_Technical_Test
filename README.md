SCTP AI Engineering Technical Entry Assessment

This repository contains my solutions for the SCTP Advanced Professional Certificate in AI Engineering technical entry assessment. The assessment covers basic programming concepts such as functions, conditionals, arrays/lists, loops, and string manipulation.

I completed the questions using Python. The tasks include a temperature converter, an inventory tracker, and a name formatting utility.

I have a background in C#, which I previously learned to create a basic GUI application. I have also dabbled in automation, ETL workflows, and Ollama. Most of my programming fundamentals were self-learned through platforms such as freeCodeCamp and Codecademy, where I built my understanding of basic coding concepts and problem-solving.
